⏰ Duration: 1 Hour

✅ What I Learned Today:
	•	How to insert images using the <img> tag:
	•	Syntax: <img src="image.jpg" alt="Description">
	•	Attributes:
	•	src → Path to the image file
	•	alt → Alternative text if image doesn’t load
	•	width, height → To resize images
	•	Types of HTML Lists:
	1.	Ordered List: <ol> – Lists with numbers
	2.	Unordered List: <ul> – Lists with bullets
	3.	Definition List: <dl>, <dt>, <dd> – Terms and definitions

What I Practiced:
	•	Added a sample image to a webpage.
	•	Created a shopping list using unordered list and a to-do list using an ordered list.
	•	Used a definition list to explain HTML term

 <!DOCTYPE html>
<html>
<head>
  <title>Images and Lists</title>
</head>
<body>
  <h2>Image Example</h2>
  <img src="flower.jpg" alt="A colorful flower" width="300" height="200">

  <h2>Unordered List - Shopping List</h2>
  <ul>
    <li>Milk</li>
    <li>Bread</li>
    <li>Eggs</li>
  </ul>

  <h2>Ordered List - Daily Tasks</h2>
  <ol>
    <li>Wake up</li>
    <li>Study HTML</li>
    <li>Exercise</li>
  </ol>

  <h2>Definition List - HTML Terms</h2>
  <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>Tag</dt>
    <dd>A code element used to define parts of a web page</dd>
  </dl>
</body>
</html>
